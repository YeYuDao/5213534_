<br/></br>### 课程介绍:<br/></br>ROS是目前主流的机器人操作系统，如同Android系统之于智能手机。虽然进入中国的时间仅短短数年，但已经受到了大多数机器人研发机构和企业的青睐和追捧。可以预见，未来它一定会成为机器人研发所必须掌握的能力。
本次讲座主要介绍ROS的起源、国内外现状、设计目标、特点以及系统结构，并带着大家实践一个简单的ROS例子。
